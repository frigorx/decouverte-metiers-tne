SchoolRoom — Guide Enseignant

Mode d'emploi pas a pas pour utiliser SchoolRoom en classe

Avant la seance
1

Preparer le dossier

Copiez le dossier complet (SchoolRoom.exe + dossier inspe/) sur le PC que vous utiliserez en classe. Cle USB ou disque dur, peu importe.

Important : Le dossier inspe/ doit etre A COTE de SchoolRoom.exe, pas a l'interieur d'un sous-dossier.
Structure attendue
Mon-Dossier/
├── SchoolRoom.exe
└── inspe/
    ├── index.html
    ├── commun/
    ├── froid/
    └── docs/
2

Tester le pare-feu (une seule fois)

Au tout premier lancement, Windows demande :

Fenetre Windows
"Voulez-vous autoriser SchoolRoom a communiquer sur le reseau ?"
→ Cliquez AUTORISER (les deux cases : prive ET public)

Si vous avez clique "Bloquer" par erreur : Parametres Windows → Pare-feu → Autoriser une application → SchoolRoom.

Sans cette autorisation, les telephones des eleves ne pourront pas se connecter.
3

Reseau WiFi

Le PC et les telephones doivent etre sur le meme reseau WiFi. Deux cas :

  • Le PC est en cable + WiFi de la salle : les eleves se connectent au WiFi de la salle. Ca marche si le reseau ne bloque pas les connexions entre appareils.
  • Le PC a un dongle WiFi : activez le "Point d'acces sans fil mobile" dans les parametres Windows. Les eleves se connectent au hotspot du PC. C'est la methode la plus fiable.
Astuce : Testez AVANT la seance avec votre propre telephone. Si ca marche avec votre telephone, ca marchera avec ceux des eleves.
Pendant la seance
4

Lancer SchoolRoom

Double-cliquez sur SchoolRoom.exe. Une fenetre noire apparait :

Ce que vous voyez
========================================
SCHOOLROOM LIGHT - DEMARRAGE
========================================
[OK] Dossiers initialises
[OK] Adresse IP locale : 192.168.0.45
[OK] Systeme operationnel

Pour arreter : Ctrl+C

Le navigateur s'ouvre automatiquement sur le Dashboard.

Ne fermez JAMAIS la fenetre noire pendant la seance. C'est le serveur. Si vous la fermez, tout s'arrete.
5

Creer la session

Sur le Dashboard, dans la section "Session" :

  • Tapez le nom de votre session (ex: 2nde TNE - Risques Pro)
  • Cliquez Valider

Ce nom apparaitra sur les telephones des eleves quand ils se connecteront.

Le nom est libre : une classe, un groupe, une formation, un intitule de TP... ce que vous voulez.
6

Afficher le QR code

Sur le Dashboard, section "QR Codes" :

  • Le QR code orange "Portail" est celui que les eleves scannent
  • Cliquez "Afficher en grand" pour le projeter au tableau
  • Ou cliquez "Telecharger PNG" pour l'imprimer a l'avance
Projetez le QR code au videoprojecteur. Les eleves le scannent avec l'appareil photo de leur telephone, ca ouvre directement la page.
7

Les eleves se connectent

Cote eleve, c'est simple :

  • L'eleve scanne le QR code avec son telephone
  • Il arrive sur la page "Bienvenue"
  • Il tape son prenom
  • Il clique "C'est parti"
  • Il arrive sur le portail des modules

A partir de la, l'eleve est autonome. Il fait les modules dans l'ordre.

8

Suivre les eleves en temps reel

Sur le Dashboard, cliquez "Vue Classe" :

  • Vert = l'eleve est actif sur l'application
  • Orange = l'eleve est en arriere-plan (il a change d'appli)
  • Rouge = l'eleve est hors-ligne (telephone eteint ou deconnecte)

La Vue Classe se rafraichit automatiquement toutes les 5 secondes.

Un eleve passe en orange ? Il est probablement sur Instagram. Un petit rappel a l'ordre suffit.
9

Voir les scores

Sur le Dashboard, cliquez "Rapport de seance" :

  • Vous voyez chaque eleve avec ses scores par module
  • Les scores sont envoyes automatiquement quand l'eleve termine un module
  • Vert = 7/10 ou plus (acquis), orange = 5-6/10, rouge = moins de 5/10
  • Cliquez "Exporter CSV" pour sauvegarder dans un tableur
Le CSV s'ouvre dans Excel. Vous avez un tableau avec tous les eleves, tous les scores, pret a archiver.
Fin de seance
10

Sauvegarder et fermer

  • Cliquez "Rapport de seance" → "Exporter CSV" pour garder une trace
  • Fermez la fenetre noire ou appuyez Ctrl+C
  • Le serveur s'arrete, les eleves sont deconnectes
Les scores sont perdus quand vous fermez SchoolRoom. Pensez a exporter le CSV AVANT de fermer.
Plan B — Si ca ne marche pas
!

Le WiFi ne marche pas

Ouvrez directement le fichier inspe/index.html depuis une cle USB sur chaque PC eleve. Les modules fonctionnent sans reseau, sans SchoolRoom. Vous perdez le suivi en temps reel, mais les eleves peuvent travailler.

!

Un telephone ne se connecte pas

  • Verifier que le telephone est sur le meme WiFi que le PC
  • Si le telephone affiche "Pas d'acces internet", cliquer "Rester connecte"
  • Si ca persiste, l'eleve travaille avec le voisin ou sur papier
!

SchoolRoom ne demarre pas

  • Verifiez que le dossier inspe/ est a cote de l'exe
  • Ouvrez http://localhost:5000/diagnostic dans le navigateur
  • Suivez les instructions affichees
Questions frequentes
Combien d'eleves peuvent se connecter en meme temps ?

SchoolRoom gere facilement 30 eleves simultanement. Au-dela, ca depend de la puissance du PC et du WiFi.

Les eleves ont-ils besoin d'installer une application ?

Non. Tout fonctionne dans le navigateur du telephone (Chrome, Safari). Rien a installer.

Est-ce que ca marche sans internet ?

Oui. SchoolRoom fonctionne en local, sans internet. Il faut juste un reseau WiFi entre le PC et les telephones (pas besoin que ce WiFi ait acces a internet).

Les donnees des eleves sont-elles envoyees quelque part ?

Non. Tout reste sur le PC du professeur. Aucune donnee ne sort du reseau local. Dispositif en logique de minimisation des donnees, sans serveur externe tiers.

Comment recuperer les scores apres la seance ?

Cliquez "Rapport de seance" puis "Exporter CSV" AVANT de fermer SchoolRoom. Le fichier CSV s'ouvre dans Excel.

Un eleve peut-il tricher en cliquant au hasard ?

Le systeme de score penalise les erreurs : premier essai correct = 2 points, deuxieme = 1 point, troisieme et plus = 0 point. Un eleve qui clique au hasard finit a 2/10.

Le bouton "Dys" ne fonctionne pas

Verifiez que le dossier assets/ contient les fichiers OpenDyslexic-Regular.otf et OpenDyslexic-Bold.otf. Ce sont les polices pour la dyslexie.
